CSS Background

CSS background property is used to define the background effects on element. There are 8 CSS background properties that affects the HTML elements:

  1. background-color
  2. background-image
  3. background-repeat
  4. background-attachment
  5. background-position
  6. background-origin
  7. background-blend-image
  8. background shorthand properties
